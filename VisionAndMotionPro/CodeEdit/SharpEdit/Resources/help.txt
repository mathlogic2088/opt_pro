
C# Editer v2.0 [墨云软件]

==========================================

帮助文档:
    
1.C# Editer v2.0是一款简单的.Net开发工具,墨云工作室最新作品.

2.上面的文本框为代码录入区[Coding],最下面为命令输入与输出区域.

3.命令系统
  
  ?         获取帮助文档信息,同“help”
  clear     清除代码区的输入
  code      获取保存到代码库中的代码
  csc       以默认方式编译当前代码(生成控制台exe程序，无输入数据，在编辑器中输出返回值)
  exit      退出
  help      获取帮助文档信息,同“?”
  hide      关闭弹出的窗体
  max       全屏
  min       最小化
  nor       恢复普通窗口
  new       新建代码
  save      保存当前代码到代码库
  test      打开测试代码示例

  [csc 编译指令](以下命令为csc命令子集，所以执行必须以csc+空格 开始)
    ?        csc帮助信息
    help     csc帮助信息
    exe      生成普通exe程序
    winexe   生成窗体exe程序
    dll      生成类库程序
    wd       使用窗口启动(普通exe程序有效)	                                                        
    arg:hi   设置参数为hi(普通exe程序有效,不能出现符号'∫',要输出空格：\s)
    show     显示编译窗口，可通过UI方式对代码进行编译执行
